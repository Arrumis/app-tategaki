# **tategaki データ構造定義書 Ver.1**

小説データおよびユーザー設定の保存形式を定義する。

## 1. ディレクトリ構造
```
data/
  ├─ favs/
  │    └─ [list_id].json         # お気に入りリスト（ユーザー別）
  ├─ novels/
  │    └─ [site_type]/           # narou / kakuyomu 等のサイト別
  │         └─ [novel_id]/
  │              ├─ info.json         # 小説の基本情報・目次
  │              ├─ ep_1.json         # 各話の本文（話数ごとに分割）
  │              ├─ ep_2.json
  │              └─ images/           # 挿絵画像
  │                   └─ [img_id].jpg
  └─ logs/
       └─ error.log              # クローラー等のエラーログ
```

## 2. 小説基本データ (`novels/[site_type]/[novel_id]/info.json`)
目次画面を生成するためのマスターデータ。

| 項目 | 型 | 説明 |
| :--- | :--- | :--- |
| `title` | String | 作品タイトル |
| `author` | String | 作者名 |
| `synopsis` | String | あらすじ（HTML/テキスト） |
| `url` | String | 元サイトの作品URL |
| `site_type` | String | `narou` または `kakuyomu` |
| `status` | String | `連載中` または `完結` |
| `total_episodes` | Number | 現在の総話数 |
| `last_checked` | String | クローラーの最終確認日時 (ISO8601) |
| `chapters` | Array | 章ごとの配列 |
| └ `chapter_title` | String | 章題 |
| └ `chapter_ep_count` | Number | その章に含まれる話数 |
| └ `episodes` | Array | 各話情報の配列 |
| 　 └ `ep_no` | Number | エピソード番号 |
| 　 └ `ep_title` | String | サブタイトル |
| 　 └ `post_date` | String | 投稿日時 |
| 　 └ `is_downloaded` | Boolean | 本文保存済みフラグ |

## 3. 各話本文データ (`novels/[site_type]/[novel_id]/ep_[no].json`)
読書画面で読み込む中身のデータ。

| 項目 | 型 | 説明 |
| :--- | :--- | :--- |
| `novel_id` | String | 親小説のID |
| `ep_no` | Number | エピソード番号 |
| `ep_title` | String | サブタイトル |
| `author` | String | 作者名 |
| `chapter_title` | String | 所属する章題 |
| `content` | String | 本文（タグ込み。なろう/カクヨム形式のHTML） |
| `images` | Array | 挿絵情報の配列 |
| └ `original_url` | String | 元の画像URL |
| └ `local_path` | String | サーバー内保存パス（`data/novels/[site_type]/[novel_id]/images/...`） |
| `prev_ep_no` | Number | 前話の番号（ない場合はnull） |
| `next_ep_no` | Number | 次話の番号（ない場合はnull） |
| `download_date` | String | ダウンロード日時 |

## 4. お気に入りリスト (`favs/[list_id].json`)
お気に入り一覧（ホーム画面）を高速表示するためのデータ。

| 項目 | 型 | 説明 |
| :--- | :--- | :--- |
| `list_id` | String | リストの識別子 |
| `novels` | Array | 登録小説の配列 |
| ├ `serial_no` | Number | リスト内での通し番号（並べ替え用） |
| ├ `novel_id` | String | 小説ID |
| ├ `site_type` | String | 元サイト種別 |
| ├ `title` | String | 作品タイトル |
| ├ `author` | String | 作者名 |
| ├ `url` | String | 元サイトURL |
| ├ `total_episodes` | Number | 全話数 |
| ├ `last_update` | String | 最新話の投稿日時 |
| ├ `bookmark` | Object | 栞情報 |
| │ └ `ep_no` | Number | 読んでいる話数 |
| │ └ `ep_title` | String | 読んでいる話のタイトル |
| │ └ `scroll_pos` | Number | 読んでいる位置（px） |
| ├ `has_new_arrival`| Boolean | 新着フラグ |
| └ `memo` | String | ユーザーメモ |
